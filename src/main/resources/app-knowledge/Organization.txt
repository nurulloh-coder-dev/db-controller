## Overview

The Organization Settings page allows users to view and manage their organization's basic information. This page displays your organization ID (required for agent configuration) and allows administrators to update the organization name.

**Purpose:** Access organization ID for agent setup and manage organization display name.

**Who uses this:** All users can view; administrators can edit organization name.

**Location:** Accessible from the sidebar by clicking "Organization" (building icon).

---

## What is the Organization Page?

The Organization page shows essential information about your organization:
- **Organization ID**: Permanent identifier needed for agent configuration
- **Organization Name**: Customizable display name shown throughout the platform
- **Activity Analytics**: Preview of upcoming analytics features (coming soon)

**Key concept:** Your organization ID never changes and is required when setting up agents to connect to your databases.

## Organization ID

### What is it?

The Organization ID is a unique, permanent identifier for your organization (UUID format).

**Example:** `a1b2c3d4-e5f6-7890-abcd-ef1234567890`

**Why it matters:**
- Required when configuring agents
- Links agents to your organization's databases
- Cannot be changed (assigned at organization creation)

### Copying Organization ID

**How to copy:**
1. Click the "Copy" button next to the organization ID
2. Button shows checkmark and "Copied!" confirmation
3. Toast notification: "Organization ID copied to clipboard"
4. Button returns to normal after 2 seconds

**Use cases:**
- Agent configuration files
- Sharing with team members setting up agents
- Integration setups

**Security note:** Organization ID is not secret - safe to share with your team. However, agents still need proper credentials to access databases.

## Organization Name

### What is it?

The Organization Name is a user-friendly label that can be customized. Unlike the ID, you can change this anytime.

**Examples:** "Acme Corporation", "Development Team", "Analytics Division"

### Viewing Organization Name

- Displayed in a highlighted box below the organization ID
- Shows current name or "Unnamed Organization" if not set
- Edit button appears on the right (for administrators)

### Editing Organization Name

**Question:** How do I change the organization name?

**Answer:** Click Edit, type the new name, and click Save.

**Step-by-step:**

1. **Click Edit button** (pencil icon)
   - Display mode switches to edit mode
   - Input field appears with current name selected

2. **Type new name**
   - Name can include letters, numbers, spaces, special characters
   - Cannot be empty

3. **Save or Cancel**
   - **Save button (green)**: Updates the name
   - **Cancel button (gray)**: Discards changes

**Save process:**
- Button shows spinner and "Saving..." text
- On success: "Organization name updated successfully" toast
- Edit mode closes, new name displays
- Other users see change after refreshing browser

**Validation:**
- Name cannot be empty
- Leading/trailing spaces automatically trimmed
- Shows error if validation fails

**Cancel:**
- Click Cancel button or press Escape key
- Returns to display mode without saving
- No changes made

### What's Affected by Name Changes

**Changes:**
- Organization name throughout UI
- Session information
- User-visible labels

**NOT affected:**
- Organization ID (never changes)
- Agent configurations (use ID, not name)
- Database connections
- User permissions

## Common Questions

### Q: What is my organization ID used for?

**A:** Your organization ID is required when configuring agents. Agents use this ID to authenticate and connect to your organization's databases.

### Q: Can I change my organization ID?

**A:** No. Organization IDs are permanent and assigned when your organization is created. This ensures stable agent configurations and reliable connections.

### Q: Is my organization ID secret?

**A:** No, it's safe to share with team members. The ID identifies your organization but cannot grant database access alone - agents still need proper credentials.

### Q: Why can't I edit the organization name?

**A:** Only administrators can edit organization settings. If you need edit access, contact your organization administrator to request permissions.

### Q: Will changing the organization name break my agents?

**A:** No. Agents use the organization ID (which never changes), not the organization name. You can safely update the name without affecting agents.

### Q: How long do organization name changes take to apply?

**A:** Changes are immediate for you. Other logged-in users see the change after refreshing their browser.

### Q: Can I have multiple organizations?

**A:** This depends on your platform configuration. Typically users belong to one organization. Contact your administrator about multi-organization setups.

## Troubleshooting

### Organization information won't load

**Solutions:**
1. Click the Retry button
2. Refresh browser (F5 or Ctrl+R)
3. Check internet connection
4. Log out and log back in
5. Contact IT support if persists

### Copy button doesn't work

**Solutions:**
1. Check browser clipboard permissions in settings
2. Update browser to latest version
3. Try different browser
4. Manually select and copy (Ctrl+C)

### Cannot save organization name

**Solutions:**
1. Ensure name is not empty
2. Check internet connection
3. Verify you have admin permissions
4. Try logging out and back in
5. Contact support with error details

### Edit button doesn't appear

**Cause:** Insufficient permissions

**Solutions:**
- Confirm you have administrator role
- Contact organization owner to request permissions
- Check Profile section to verify your role

### Name change doesn't show for other users

**Solutions:**
1. Ask other users to refresh their browsers
2. Verify your change saved successfully (check for success toast)
3. Check if change appears when you refresh
